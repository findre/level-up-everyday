## 存储类别
```c
int entity = 3;
```
1. 被存储的每一个值都占有一定的物理内存
2. C语言把这样的一块儿内存称为对象(object)
3. 对象可以存储一个或多个值
4. 对象根据实际值具有适当大小
5. entity 在C语言中称为标识符(identifier), 用来指定特定对象内容。
